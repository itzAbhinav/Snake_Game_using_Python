# 🐍 Snake Game (Python Turtle)

A simple Snake Game built using Python’s built-in turtle module. Control the snake, eat food, grow longer, and avoid collisions!

# 🎮 Features
Classic snake movement (Up, Down, Left, Right)
Food spawning at random positions
Score tracking system
Snake grows as it eats food
Collision detection:
Wall collision
Self collision
Increasing difficulty (speed increases as score increases)

# 🛠️ Technologies Used
Python 3
turtle (for graphics)
random (for food placement)
time (for game speed control)

#  📸 Gameplay Overview

The snake starts at the center of the screen.
Eat the pink food to grow and increase your score.
Each time you eat food:
Score increases
Snake length increases
Game speed slightly increases
Game ends if:
Snake hits the wall
Snake collides with itself
